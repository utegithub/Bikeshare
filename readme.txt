Bikeshare data:
The following program shows you interesting statistics about Bikesharing.

Getting started:
Store bikeshare.py and the csv files: chicago.csv, new_york_city.csv and washington.csv in the same folder.
Python is required to run this program.
Check if python is installed by opening the command line tool and entering python -V
If python is installed, the program can be run from the command line tool.

How to use the program:
To start a city must be selected.
Valid input: New York City, Washington, Chicago
In second and third step, month and day can be selected. Entering “All” will not filter based on month or day
Valid input for month:
January, February, March, April, May, June, All
Valid input for day:
Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday
